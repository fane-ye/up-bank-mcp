# Changelog

All notable changes to Up Bank MCP are documented here.

## [v1.0.0] — 10 May 2026

Initial release.

### Added
- 20 MCP tools covering the full Up Bank API: accounts, transactions, categories, tags, attachments, and webhooks
- Cursor-based auto-pagination — large result sets are fetched across multiple API pages automatically
- Distributed as an `.mcpb` bundle for one-click install into Claude Desktop
- Secure token handling via the OS keychain — the Up Bank Personal Access Token is never written to plaintext config
